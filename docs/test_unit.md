- 토양관 - 미니게임 - 결과 화면에서 박물관 배경음악하고 겹치는 오류

- 축구장 - 축구 체험 후 다른 곳에서 공을 차는 소리가 들림
- 축구장 - 나올 때 앵글
- 신재생에너지관 풍력발전 설명글 어두움.

- 신재생에너지관 폐기물에너지 설명글 어둡
